# Конструктор рассылок


## Краткое описание
Веб-приложение для создания простых Email рассылок. Приложене используется для упрощения процесса верстки рассылок и дальнейшей загрузки верстки во внутреннюю систему рассылок. Все картинки находятся на внутреннем ресурсе в элементы стиля определены брендбуком ДИТ Москвы.

## Функционал
* Добавление/удаление контентных блоков
* Добавление/редактирование/удаление элементов внутри контентных блоков
* Изменение стилей текста (начертание, цвет)
* Добавление/удаление ссылок
* Импорт рассылки
* Экспорт HTML рассылки

## Ограничения
* Рассылки не имеют темной темы
* Редактирование картинок происходит через изменение ссылок на внутренний ресурс (т.к. у ресурса нет API)
* Верстка кнопок ломается при использовании в десктопном приложении Outlook

## Технологический стек
* Язык: JavaScript/Typescript
* Фреймворк: Vue js (Vue 3 Composition Api)
* Стейт менеджер: Pinia

## Используемые библиотеки
* [TipTap](https://tiptap.dev/) - Редактор текста для содержимого блоков.
* [Nano ID](https://www.npmjs.com/package/nanoid) - Для генерации уникальных, коротких ID.
* [Lodash](https://lodash.com/) - Утилита для упрощения работы с аассивами и объектами.
