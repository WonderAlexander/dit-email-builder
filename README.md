# Конструктор рассылок


## Краткое описание
Веб‑приложение для создания и быстрой вёрстки простых email‑рассылок с последующей загрузкой во внутреннюю систему. Все изображения хранятся на внутреннем ресурсе, стили соответствуют брендбуку ДИТ Москвы.

## Функционал
* Добавление/удаление контентных блоков
* Добавление/редактирование/удаление элементов внутри контентных блоков
* Изменение стилей текста (начертание, цвет)
* Добавление/удаление ссылок
* Импорт рассылки
* Экспорт HTML рассылки

## Ограничения
* Тёмная тема не поддерживается
* Изображения редактируются заменой ссылок на внутренний ресурс (нет доступного API)
* Вёрстка кнопок может ломаться в Outlook Desktop из-за особенностей рендеринга

## Архитектура
**Стек:**
* Vue 3 (Composition API)
* TypeScript/JavaScript
* Pinia <br>

**Библиотеки:**
* [TipTap](https://tiptap.dev/) - Редактор текста для содержимого блоков.
* [Nano ID](https://www.npmjs.com/package/nanoid) - Для генерации уникальных, коротких ID.
* [Lodash](https://lodash.com/) - Утилита для упрощения работы с массивами и объектами.

**Структура:**

├─ src/                        
    ├─ components/           — переиспользуемые UI-компоненты (для фронта)    
    │  ├─ Editors/           — Компоненты редактирования элементво рассылки  
    │  ├─ Email/             — Компоненты верстки рассылки  
    │  ├─ Icons/             — Компоненты иконок  
    │  ├─ ModalContent/      — Компоненты модальных окон  
    │  ├─ UI/                — Компоненты UI  
    ├─ routes/               — Роуты 
    ├─ stores/               — Pinia стор
    ├─ types/                — Типизация  
    ├─ utils/                — Утилиты (функции. данные)  
    ├─ views/                — Представления
